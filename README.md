# spring-boot-examples

### 技术选型
- 核心框架：SpringBoot 1.5.7,SpringCloud Edgware.RELEASE
- 持久层框架：Mybatis 3.4.5,Mybatis-plus 2.1.6
- 数据库连接池：druid 1.1.6
- 缓存框架：Redis 
- 日志管理：Log4j
- 前端框架：Layui+Jquery+Freemarker模板
